Tag VS Element VS Attributes

Tag

Tag are those which comes in between < angularbrackets >.

90% tags are come with Starting And Closing Tag

for eg. h1, p, head, html, body etc

10% Tags Are self closing tag or void Tag or Empty tag.

for eg. br, hr, img, input, meta etc

Element

Start Tag - Some Content - End Tag

Attributes

They Provide Some Additional Information Which Change The behaviour of Element. Always comes in Start Tag
